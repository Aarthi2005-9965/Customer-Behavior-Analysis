Customer Behavior Analysis for Business Decision Making

Project Overview:-
This project analyzes customer behavior and predicts customer churn using Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-Learn.

Objectives:-
- Analyze customer behavior
- Identify churn patterns
- Visualize business insights
- Predict customer churn using machine learning

Tools Used:-
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

Results:-
- Churn analysis completed
- Business insights generated
- Random Forest model trained
- Customer churn predicted successfully
